# Flying Cat Media Player

## Features:
* ### Play Audio Files
	* Play Audio files from your PC with the following features: 
		* Play/Pause
		* Skip/Back
		* Forward/Back 15 Seconds
		* MediaStream
* ### Play Video Files
	* Play video files from your PC with the following features:
	 	* Play/Pause
		* Skip/Back
		* Forward/Back 15 Seconds
		* MediaStream
		* Fullscreen
		* PiP
